# SAP setup checklist

This document describes manual setup steps that still need to be completed in an SAP account.

1. Create or access an SAP account.
2. Create an SAP BTP trial account.
3. Create an ABAP Environment trial.
4. Install Eclipse and ABAP Development Tools.
5. Create an ABAP Cloud project.
6. Create a package named ZNA_FINANCE.
7. Import the draft ABAP source objects.
8. Activate the objects after required adjustments.
9. Create a service binding for the OData UI service.
10. Preview the Fiori Elements app.
11. Capture genuine screenshots from the activated environment.

These steps are manual and remain unverified in this repository.
