# OData V4 UI service binding draft

This file documents the manual steps for creating an OData V4 UI service binding in Eclipse ADT.

1. Create an ABAP Cloud project.
2. Create or activate the service definition for ZNA_SD_INVOICE.
3. In Eclipse ADT, create a service binding for the OData V4 UI.
4. Choose the draft service definition.
5. Generate the binding and preview the app.
6. Capture screenshots only after the binding is working in a real SAP system.

This source must be imported, adjusted where required, compiled, and activated in an SAP BTP ABAP Environment before it can be described as working SAP code.
